# Identificación de Empleados Críticos mediante Análisis Organizacional y Análisis de Sentimientos

## Descripción

Proyecto desarrollado en Python para analizar información organizacional utilizando técnicas de minería de datos, visualización y análisis de sentimientos.

## Objetivos

- Analizar satisfacción laboral.
- Evaluar carga de trabajo.
- Analizar rotación de colaboradores.
- Aplicar análisis de sentimientos mediante VADER.
- Identificar empleados críticos mediante un índice de riesgo.

## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Faker
- Matplotlib
- Seaborn
- Plotly
- VADER Sentiment

## Principales variables derivadas

- tenure_years
- overloaded
- sentiment_score
- sentiment_label
- risk_index
- risk_level
